# SERP Replacement Logic

Before diagnosing a content decline, ask whether traffic disappeared or moved.

## Process

1. Select important lost queries.
2. Check current top results.
3. Classify replacements:
   - direct competitors
   - independent review sites
   - forums/community results
   - official docs
   - media/publishers
   - SERP features or AI answers
4. Compare replacement pages to the lost page:
   - answer set
   - criteria
   - firsthand proof
   - product promotion
   - freshness
   - time to value
   - page type
5. Decide whether demand likely shrank, rankings transferred, or both.

## Conclusions To Avoid

- "AI killed it" without query-level evidence.
- "Google punished topical overreach" when direct product-category queries also died.
- "Backlinks caused it" when links improved or competitors with similar junk links still rank.
- "Domain authority is enough" when lower-authority pages replaced the site.

## Output

For each query or cluster:

- Former page
- Current winners
- Replacement type
- What winners satisfy better
- What evidence is missing
- Implication for rewrite/prune/recovery
