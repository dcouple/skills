# Portfolio Action Matrix

Use this matrix to turn audit findings into actions.

| Condition | Action |
|---|---|
| Good topic, bad execution | Rewrite |
| Good page, stale facts/screenshots/tools | Refresh |
| Multiple pages satisfy same intent | Merge |
| No product fit, no authority, no useful traffic | Prune |
| Thin template variant with better parent page | Redirect or consolidate |
| Useful page with technical/schema issue | Fix |
| SERP changed to a different page type | Reposition |
| AI/demand loss but commercial value remains | Reframe toward action or decision |
| Legal/compliance/brand risk | Manual review |

## Sequencing

1. Stop publishing more of the harmful pattern.
2. Fix technical blockers that affect many pages.
3. Prune or consolidate clear low-value pages.
4. Rewrite high-value core-topic pages.
5. Refresh stale pages with retained demand.
6. Publish new pages only after standards are updated.

## Output Columns

- URL
- Section
- Topic
- Intent
- Product fit
- Evidence
- Recommended action
- Priority
- Manual-review reason
