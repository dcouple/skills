# Editorial Voice

Authoritative voice is not confidence theater. It comes from specificity, evidence, and useful tradeoffs.

## Improve Voice By

- Replacing generic claims with specific observations.
- Naming who a recommendation is and is not for.
- Admitting tradeoffs plainly.
- Using firsthand language only when there is firsthand evidence.
- Showing criteria before conclusions.
- Removing filler intros and repeated setup.
- Avoiding fake neutrality when the page is owned by a vendor.
- Making limitations actionable.

## Weak Voice Patterns

- "Powerful, seamless, robust" without evidence.
- Every tool described with the same template.
- Pros and cons that could apply to any product.
- Long paragraphs that repeat the page title.
- Claims about competitors without source, screenshot, test, or methodology.
- Overexplaining obvious terms while skipping the decision criteria.

## Rewrite Pattern

Weak:
`Tool X is a powerful solution for teams looking to improve productivity.`

Stronger:
`Tool X is strongest for teams that already manage work in spreadsheets and need lightweight automation. It is weaker for cross-functional planning because permissions and dependencies are limited.`

## Final Check

If a sentence could appear unchanged on a competitor's page, make it more specific or cut it.
